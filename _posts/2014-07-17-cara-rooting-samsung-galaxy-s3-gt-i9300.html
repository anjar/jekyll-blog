Banyak yang bertanya apa sih yang dimaksud root / rooting dalam smartphone android itu?. Root adalah user account di Linux (Android dikembangkan berbasis Linux) yang memiliki segala akses. User Root dapat meng-edit apapun di system. Demi alasan keamanan, user biasa tidak memiliki semua akses di system. Saat kita melakukan rooting di SGS3 maka kita akan mendapatkan akses untuk membaca dan menulis di system dimana sebelumnya tidak dimiliki dan kita juga dimungkinkan untuk menjalankan beberapa perintah didalam Terminal.

Sebelum melakukan rooting, ada baiknya di baca terlebih dulu keuntungan dan kerugian Rooting di Samsung Galaxy S3 GT-I9300.

Keuntungan Root

  • Dengan Root kita dapat melakukan uninstall aplikasi bawaan vendor yang tidak dapat di uninstall apabila belum di root.
  • Backup system dan aplikasi, sehingga memudahkan user backup 1 to 1 dengan smartphone yang sama. Contoh aplikasi backup salah satunya dalah  Titanium Backup
  • Dengan kondisi root kita dapat melakukan instalasi Custom ROM.
  • Overclock CPU. Mempunyai akibat 2 sisi mata uang, di satu sisi meningkatkan performa namun disisi lain memuat konsumsi baterei lebih boros (warning overclock CPU harus dijalankan secara hati-hati karena dapat merusak hardware akibat panas yang terlalu berlebihan).
  • Memungkinkan install aplikasi pada memory card.
  • Ad-Free (tdk ada iklan), GPS, GPU (dgn Tegra, Snapdragon atau POwerVR - aplikasi chainfire3D), blok akses internet utk beberapa aplikasi, unlock Sim Card untuk SGS3 dengan carrier Orange, O2, T-mobile, Three & TMN,. Banyaaak deh pokoke tweak, tweak...and tweak


Kerugian Root

  • Menyebabkan hilang garansi. Hal ini sering dipermasalahkan oleh pihak vendor, namun tidak menjadi masalah karena smartphone dapat dikembalikan ke kondisi sebelum di root melalui proses Unroot.
  • Keamanan. Walaupn masih diperdebatkan, dimasukkan aja yaa. Dengan membuka akses sehingga dapat terinfeksi malware, spam atau virus. Dapat diantispasi dengan menginstall antivirus atau aplikasi remove malware, dan mengintall aplikasi yang telah dipercaya dan source yang jelas.


Langkah pertama untuk rooting, silahkan unduh file untuk rooting Samsung Galaxy S3  CF-Root-SGS3, kemudian extrak file tersebut, ingat cukup sekali saja, sampai ada file dengan extensi .tar.

Kemudian silahkan unduh Odin di http://www.android.gs/download-odin-1-85/ . Setelah itu, extrak file Odin yang telah di download tadi kemudian buka aplikasi tersebut. matikan Samsung Galaxy S3 GT-I9300 dan boot ke dalam mode download dengan menekan tombol "Volume Bawah" + "Power" + "Home" secara bersamaan. kemudian sambungkan Samsung Galaxy S3 GT-I9300 dengan komputer menggunakan Kabel USB. Jika Samsung Galaxy S3 GT-I9300 sudah tersambung dengan Odin maka akan muncul warna kuning dengan tulisan "Com X", dimana X adalah Nomor dari port USB tergantung PC masing-masing.
