Random Sort Order Apache SOLR adalah salah satu cara untuk menghasilkan result random di apache solr seperti halnya "order by rand()" di MySQL adalah dengan menambahkan dynaimc field random di schema.xml.

Pertama tambahkan type field random berikut di schema.xml


Kemudian tambahkan field random di schema.xml


Kemudian restart solr. Untuk mendapatkan hasil random dapat dengan mengorder by random field , contoh sort=random_1234 , buka browser lalu coba untuk random lalu akses di browser

http://localhost:8983/solr/select?q=*%3A*&sort=random_121+desc&wt=xml&indent=true
